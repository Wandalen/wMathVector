### Що таке вектор
У математиці вектор - індексована сукупність чисел або інших математичних обєктів( х1,х2,...xn ), яка має певні властивості, що дозволяють виконувати над вектором різні математичні операції: додавання, віднімання, множення на скаляр тощо. У вигляді вектору можна позначати положення, швидкість, прискорення, і тп. Зручність вектору полягає в можливості оперувати великою кількістю чисел як одним обєктом. В інформатиці вектор прийнято представляти у вигляді одновимірного масиву.

### wMathVector
wMathVector - пакет, що реалізує вектор. wMathVector дозволяє оперувати векторами через обгортки до даних.
Обгортки не містять безпосередньо даних, а лише метадані такі, як наприклад, початок, кінець вектору в масиві, лінк на масив із даними.
Доступ до інтерфейсу для роботи із векторами здійснюється через неймспейси: ```wTools.vector``` та ```wTools.avector```.

### wMathVector

### Створення обгортки
wTools.vector.from свторює обгортку вектору із масиву.

```javascript
var a1 = [ 1, 2, 3 ];
var v1 = wTools.vector.from( a1 );
console.log( 'v1: ', v1 );
```
Вивід:
```
v1:  VectorFromArray { _vectorBuffer: [ 1, 2, 3 ] }
```
Змінна ```v1``` стала обгорткою вектору, тепер над нею можна здійснювати векторні операції.

### Довжина вектору

Довжина вектору міститься в полі ```length``` обгортки.

```javascript
var a1 = [ 1, 2, 3 ];
var v1 = wTools.vector.from( a1 );
console.log( 'v1.length: ', v1.length )
console.log( 'a1.length: ', a1.length )
```
Вивід:
```
v1.length:  3
a1.length:  3
```
Довжина вектору ```v1``` рівна кількості елементів у контейнері ```a1```.

### Доступ до даних

#### Метод toStr повертає вміст вектору у вигляді стрічки.

```javascript
var a1 = [ 1, 2, 3 ];
var v1 = wTools.vector.from( a1 );
console.log( 'v1.toStr(): ',  v1.toStr() );
```
Вивід:
```
v1.toStr():  1.000 2.000 3.000
```
#### Метод eGet повертає значення елемента по індексу.

```javascript
var a1 = [ 1, 2, 3 ];
var v1 = wTools.vector.from( a1 );
console.log( 'v1.eGet( 2 ): ', v1.eGet( 2 ) );
```
Вивід:
```
v1.eGet( 2 ):  3
```
Виклик ```v1.eGet( 2 )``` повертає значення елемента вектору ```v1``` з індексом 2.

### Зміна даних

#### Метод eSet змінює значення елемента по індексу.

```javascript
var a1 = [ 1, 2, 3 ];
var v1 = wTools.vector.from( a1 );
v1.eSet( 0, 5 );
console.log( 'v1.toStr(): ',  v1.toStr() );
```
Вивід:
```
v1.toStr():  5.000 2.000 3.000
```
Виклик ```v1.eSet( 0, 5 )``` перевизначає перший елемент вектору ```v1``` числом 5.

#### Результат операцій над вектором зберігається у його контейнері.

```javascript
var a1 = [ -1, -1, -1 ];
var v1 = wTools.vector.from( a1 )
wTools.vector.abs( v1 );
console.log( 'v1.toStr(): ',  v1.toStr() );
console.log( 'a1: ', a1 );
```
Вивід:
```
v1.toStr():  1.000 1.000 1.000
a1:  [ 1, 1, 1 ]
```
Метод ```abs``` замінює відємні значення вектору ```v1``` додатніми та оновлює вміст контейнера ```a1```.

### Виконання математичних операцій

#### Метод add додає вектори.

```javascript
var a1 = [ 1, 1, 1 ];
var a2 = [ 1, 1, 1 ];
var v1 = wTools.vector.from( a1 );
var v2 = wTools.vector.from( a2 );
var result = wTools.vector.add( v1, v2 );
console.log( 'result.toStr(): ', result.toStr() );
```
Вивід:
```
result.toStr():  2.000 2.000 2.000
```
Змінна ```result``` містить вектор із результатом додавання векторів ```v1``` та ```v2```.

#### Метод add додає безліч векторів.

```javascript
var a1 = [ 1, 1, 1 ];
var a2 = [ 1, 1, 1 ];
var a3 = [ 1, 1, 1 ];
var v1 = wTools.vector.from( a1 );
var v2 = wTools.vector.from( a2 );
var v3 = wTools.vector.from( a3 );
var result = wTools.vector.add( v1, v2, v3 );
console.log( 'result.toStr(): ', result.toStr() );
```
Вивід:
```
result.toStr():  3.000 3.000 3.000
```
Змінна ```result``` містить вектор із результатом додавання трьох векторів: v1,v2,v3.

#### Метод add використовує для зберігання результату вектор, що переданий першим аргументом

Збережемо результат додавання у другому векторі.

```javascript
var a1 = [ 1, 1, 1 ];
var a2 = [ 1, 1, 1 ];
var v1 = wTools.vector.from( a1 );
var v2 = wTools.vector.from( a2 );
wTools.vector.add( v2, v1 );

console.log( 'v2.toStr(): ', v2.toStr() );
```
Вивід:
```
v2.toStr():  2.000 2.000 2.000
```
Вектор ```v2``` перезаписано результатом додавання векторів ```v1``` та ```v2```.

#### Метод add повертає результат у новому векторі, якщо перший аргумент null

```javascript
var a1 = [ 1, 1, 1 ];
var a2 = [ 1, 1, 1 ];
var v1 = wTools.vector.from( a1 );
var v2 = wTools.vector.from( a2 );
var result = wTools.vector.add( null, v1, v2 );

console.log( 'result.toStr(): ', result.toStr() );
console.log( 'v1.toStr(): ', v1.toStr() );
console.log( 'v2.toStr(): ', v2.toStr() );
```
Вивід:
```
result.toStr():  2.000 2.000 2.000
v1.toStr():  1.000 1.000 1.000
v2.toStr():  1.000 1.000 1.000
```
Змінна ```result``` містить нову обгортку із результатом додавання векторів ```v1``` та ```v2```.
Вектори ```v1``` та ```v2``` залишились без змін.

#### Метод add векторизує скаляр

Векторизація скаляру - створення вектору необхідної довжини та заповнення значенням скаляру.

```javascript
var a1 = [ 1, 1 ];
var v1 = wTools.vector.from( a1 );
wTools.vector.add( v1, 3 );
console.log( 'v1.toStr(): ', v1.toStr() );
```
Вивід:
```
v1.toStr():  4.000 4.000
```
Виклик ```wTools.vector.add( v1, 3 );``` перетворить скаляр ```3``` у вектор із довжиною вектору ```v1``` та заповнить його значенням скаляру.
В результаті отримаємо вектор ( 3, 3 ), який буде додано до вектору ```v1```. Такий метод виключає необхідність ручного створення векторів із однаковим вмістом.

#### Метод add працює лише з векторами однакової довжини

```javascript
var a1 = [ 1, 1 ];
var a2 = [ 1, 1, 1 ];
var v1 = wTools.vector.from( a1 );
var v2 = wTools.vector.from( a2 );
wTools.vector.add( v1, v2 );
```
Виклик ```wTools.vector.add( v1, v2 );``` завершиться із помилкою тому, що всі вектори повинні мати однакову довжину.

*Методи sub,mul,div працюють за аналогічним із ```add``` принципом, відмінністю є лише операція,яку вони виконують із вектором.*

### Виконання операцій із вектором через інтерфейс wTools.avector

```wTools.avector``` виконує операції із вектором аналогічно до ```wTools.vector```, про те його методи використовують в якості аргументів звичайні масиви.

#### wTools.avector.add додає вектори у вигляді масивів

```javascript
var a1 = [ 1, 1, 1 ];
var a2 = [ 1, 1, 1 ];
wTools.avector.add( a1, a2 );
console.log( 'a1: ', a1 );
```
Вивід:
```
a1:  [ 2, 2, 2 ]
```
Результат додавання записано у масиві ```a1```.
